What is Queue?

Queue is a Linear Data Structure that follows FIFO (First In First Out). The element inserted first will be removed first.

Advantages

  • Fast insertion at Rear
  • Fast deletion at Front
  • Useful in scheduling systems
  • Easy FIFO processing

Disadvantages

  • No random access
  • Searching takes O(n)
  • Can waste memory in array implementation
