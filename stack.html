What is Stack?

Stack is a Linear Data Structure that follows LIFO (Last In First Out). The last inserted element will be removed first.

Advantages

  • Fast Insertion
  • Fast Deletion
  • Easy Memory Management
  • Used in Function Calls

Disadvantages

  • No Random Access
  • Overflow Possible
  • Limited Size
