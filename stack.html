What is a Stack?

Stack is a linear data structure that follows LIFO (Last In First Out). The last inserted element is removed first.

Advantages

  • Fast insertion
  • Fast deletion
  • Easy memory management
  • Used in function calls

Disadvantages

  • No random access
  • Overflow possible
  • Limited size

Applications

  • Browser History
  • Undo Operations
  • Function Calls
  • Expression Evaluation
