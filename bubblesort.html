What is Bubble Sort?

Bubble Sort is a simple sorting algorithm that repeatedly compares adjacent elements and swaps them if they are in the wrong order. After each pass, the largest element moves to its correct position.

Real-Time Applications

  • Educational learning and algorithm visualization.
  • Small datasets where simplicity is preferred.
  • Teaching sorting fundamentals.
  • Used in introductory programming courses.

Time & Space Complexity

Case Complexity
Best Case O(n)
Average Case O(n²)
Worst Case O(n²)
Space Complexity O(1)

Java Syntax

for(int i = 0; i < n - 1; i++){

    for(int j = 0; j < n - i - 1; j++){

        if(arr[j] > arr[j + 1]){

            int temp = arr[j];
            arr[j] = arr[j + 1];
            arr[j + 1] = temp;

        }

    }

}
