What is StringBuffer?

StringBuffer is a mutable sequence of characters. It is synchronized and thread-safe, making it suitable for multi-threaded applications.

What is StringBuilder?

StringBuilder is a mutable sequence of characters. It is not synchronized and is faster than StringBuffer. It is mainly used in single-threaded applications.

Java Syntax


StringBuffer sb1 =
new StringBuffer("Hello");

sb1.append(" Java");

sb1.insert(5," World");

StringBuilder sb2 =
new StringBuilder("Hello");

sb2.append(" Java");

sb2.reverse();

Common Properties

  • Mutable
  • Dynamic in Size
  • Supports Character Manipulation
  • Provides Built-in Methods
  • Can Modify Existing Data
