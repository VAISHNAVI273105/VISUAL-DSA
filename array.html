What is an Array?

Array is a linear data structure used to store multiple elements of the same datatype in contiguous memory locations.

Advantages

  • Fast access using index
  • Easy traversal
  • Simple implementation
  • Memory efficient

Disadvantages

  • Fixed size
  • Insertion is costly
  • Deletion requires shifting
  • Memory wastage possible

Applications

  • Searching
  • Sorting
  • Matrices
  • Dynamic Programming
